Magpie is installed.

Find it in /Applications.

Before your first launch:

  • Install yap: brew install yap
  • Make sure Claude Code is installed and authenticated

First launch:

  • Open Magpie from Applications
  • Choose your output folder
  • Grant Microphone access when prompted
  • Optionally grant Screen & System Audio Recording for capturing both sides of calls

Start recording from anywhere with Cmd+Shift+R.
