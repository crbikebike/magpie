Welcome to Magpie

A meeting recorder that transcribes and summarizes with Claude.

This installer will place Magpie.app in your /Applications folder.

After installing, you'll need to:

  • Install yap for transcription: brew install yap
  • Have Claude Code installed and authenticated
  • Grant Microphone permission on first launch
  • Choose an output folder

macOS 14.4 or later required.
